# Estudo Dirigido 5 – Controller Hardware-in-the-Loop (C-HIL)

Este repositório contém a implementação manual do controlador PR desenvolvida no **Code Composer Studio (CCS)** para o DSP **TMS320F28379D**, utilizada no Estudo Dirigido 5 da disciplina.

## Conteúdo do repositório

- Código-fonte da implementação manual do controlador PR em C;
- Configuração dos periféricos utilizados (ADC e ePWM);
- Arquivos de configuração do projeto do CCS.

## Observação

Os arquivos referentes ao modelo desenvolvido no **PLECS** e ao projeto do **Typhoon HIL** não estão incluídos neste repositório, foram encaminhados separadamente em formato compactado (.zip).
